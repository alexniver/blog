Syntax Highlighting

Feb 8, 2017 3 min.

Syntax highlighting works out-of-the-box via Hugo. Set the highlighter style in config.toml: pygmentsstyle = "<style>" For dark highlighter themes, you should rebuild capsule CSS with build/extra/syntax.sass. Uncomment it from the capsule.sass file and run gulp. Without that file, the background color will default to Bulma’s light background-color, and light colored elements meant to be displayed against dark backgrounds from dark themes will be hard to read. Use ``` braces to add code blocks.
